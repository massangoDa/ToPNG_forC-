・インストール方法
　配布ファイル(topngsetup.exe)自体がインストーラーです。
　それをダブルクリックして開いてください。

・アンインストール方法
　削除したくなった場合は、コントロールパネルの中の「アプリケーションの
　追加と削除」のアイコンを開いて、一覧からtopngを見つけ、選択してくだ
　さい。

・使い方
　デスクトップにあるtopng.exeまたは、C://TOPNG/ 内の、topng.exeをダ
　ブルクリックしてください。
